# 🏡 Property Price Prediction App

This project predicts property prices using machine-learning models trained on the [Kaggle House Prices dataset](https://www.kaggle.com/c/house-prices-advanced-regression-techniques).

## 🧩 Features

* Linear Regression & Random Forest models
* Clean Streamlit UI
* Interactive inputs for area, bedrooms, bathrooms, year built, neighborhood
* Real-time predictions

## 🧠 Tech Stack

Python · Pandas · Scikit-learn · Streamlit · Joblib

## 🚀 How to Run

1. Clone the repo or download the zip.
2. Install requirements:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:

   ```bash
   streamlit run src/app.py
   ```
